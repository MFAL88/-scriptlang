## 🚀 Pourquoi ScriptLang ?

Parce que `personnage héros = "Alice"` est plus naturel que `var hero = "Alice"` quand on raconte des histoires !

ScriptLang révolutionne l'apprentissage de la programmation en utilisant le vocabulaire de l'écriture :

- 🎭 **`personnage`** au lieu de `variable`
- 📚 **`chapitre`** au lieu de `fonction` 
- ✍️ **`écrire`** pour afficher du texte
- 🔄 **`répéter X fois`** pour les boucles
- 🤔 **`si/sinon`** pour les conditions
- 📖 **`nouveau_chapitre`** pour structurer le code

---

## 💡 Exemple Concret

```scriptlang
// Une histoire interactive en ScriptLang
personnage nom = demander("Quel est votre nom ?")
personnage âge = demander("Quel âge avez-vous ?")

si âge >= 18 alors
    écrire("Bonjour " + nom + ", bienvenue dans cette aventure adulte !")
sinon
    écrire("Salut " + nom + ", prêt pour une aventure jeunesse ?")

répéter 3 fois
    écrire("🌟 Que l'aventure commence !")

nouveau_chapitre fin_histoire()
    écrire("Merci d'avoir lu notre histoire !")
```
